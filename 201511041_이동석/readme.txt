Readme 파일: 데이터출처, 추가 패키지 설치 필요한 경우 설명 작성

1. 데이터 출처 : 
저는 http://taas.koroad.or.kr/api/selectDeathDataSet.do
사고별 사망자 정보 csv 파일을 얻을 수 있었습니다.
그렇지만, 기존의 '12_19_death.csv를 읽는 과정에서 오류가 발생하였고, 구글링 결과 csv 파일의 인코딩 방식이 달라서 생김 문제임을 알았습니다.
그래서 저는 excel을 이용해  12_19_death.csv를 csv-utf-8형식으로 12_19_death_utf8.csv로 저장함으로써 해결하였습니다.

2. 추가 패키지, 파일 설치 - 필요없습니다.
파이 플롯을 그리기 위해서 한글설정 파일이 필요했습니다.
파일 경로에 있는 NanumFontSetup_TTF_SQUARE 를 그대로 이용하였고,
상대경로를 이용하였기 때문에 추가 패키지, 파일 설치는 필요없습니다.

3.추가정보 : 머신러닝 입력, 목표 변수 설명.
저는 
입력 변수(독립변수)를
당사자종별_1당_대분류,	도로형태_대분류,	발생지시도,	사상자분류,	주야분류 로 하였습니다.
